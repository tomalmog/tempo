# Example: Single Task Prompt

Build a full-stack blog application with the following requirements:

## Backend
- Express.js with TypeScript
- PostgreSQL database with Prisma ORM
- JWT authentication (register, login, logout)
- RESTful API for posts (CRUD)
- Input validation with Zod
- Rate limiting and security headers

## Frontend
- React with TypeScript and Vite
- Tailwind CSS for styling
- React Router for navigation
- React Query for data fetching
- Protected routes for authenticated users
- Responsive design

## Features
1. User registration and login
2. Create, edit, delete blog posts (authenticated)
3. View all posts (public)
4. View single post (public)
5. User profile page
6. Markdown support for post content

## Project Structure
- `/backend` - Express API
- `/frontend` - React app
- Docker Compose for local development

Please implement this step by step, testing each feature as you go.
